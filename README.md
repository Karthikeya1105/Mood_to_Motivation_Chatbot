Mood-to-Motivation Bot

A web-based chatbot that detects the user’s mood from text input and generates personalized motivational responses. Built as part of Day-5 Hackathon at Codegnan NLP Workshop.

🚀 Features

Detects mood/emotions from user input using Hugging Face models

Generates motivational responses using GenAI API

Interactive UI powered by Streamlit

Real-time mood-to-motivation conversion

🛠️ Tech Stack

Python

Streamlit – Frontend web app

Hugging Face Transformers – NLP emotion detection

GenAI API – Dynamic motivational text generation

📂 Project Structure
├── Mood_motivation.py     # Main Streamlit app
├── README.md              # Project documentation

🎯 Usage

Enter your mood or a sentence describing how you feel.

The model analyzes your mood.

Get an instant motivational message tailored to your emotional state.

📌 Example

Input: "I am feeling really stressed today."
Output: "Take a deep breath! Challenges only make you stronger. You’ve got this 💪."
